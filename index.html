Things I've Made

Tembo

Anyone who has ever worked on a team with a shared email inbox knows how painful it can be to delegate work efficiently while staying up-to-date with all of the different tasks. Tembo was built to help alleviate some of that pain. It's a Chrome extension for Gmail that provides shared thread histories, labels, and alerts — all of which update in real-time across different individual Gmail accounts. Tembo also features a dashboard that team members can use to customize their labels and email templates.
